## What data can I export?
### Engagement Data
- **Users Twitter Accounts**: Details about each TG user's Twitter account(s).
- **Raids**: Details about each raid including tweet information, engagement targets, and actual engagement counts
- **Likes**: Information about users who liked raided tweets
- **Retweets**: Information about users who retweeted raided tweets
- **Bookmarks**: Information about users who bookmarked raided tweets
- **Replies**: Information about users who replied to raided tweets
- **Smart Replies**: Details about automated replies made through the Smart Reply system

### Aggregation Data
- **Total Counts**: Overall statistics including total raids, likes, retweets, bookmarks, and replies
- **Raids by Source**: Breakdown of raids created by admins vs AutoRaid
- **Top Engagers**: Lists of most active users in different categories:
  - Top Likers
  - Top Retweeters
  - Top Bookmarkers
  - Top Repliers

### Date Range Options
- **From**: Required start date for the export period
- **To**: Optional end date for the export period

The export will be generated as an Excel (.xlsx) file with separate worksheets for each selected data type.
