### View Leaderboard Rankings

Use the `/leaderboard` command to open the Leaderboard page where you can:

- View top engaging users by different metrics
- Switch between engagement types via tabs:
  - **Likes**: Top users by like count
  - **Retweets**: Top users by retweet count
  - **Replies**: Top users by reply count
  - **Bookmarks**: Top users by bookmark count

### Filter & Search

Easily find specific users or view different time periods:

- **Search**: Look up users by their name or username
- **Timeframes**: Filter rankings by:
  - Daily (Start of day to end of day)
  - Weekly (Monday to Sunday)
  - Monthly (1st to last day)
