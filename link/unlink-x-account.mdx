## Unlink X Account via `/link`
You're able to easily unlink your X Account from the bot if you no longer want to use it for engaging on raids.

### Instructions

1. Type `/link` within a Telegram Group that has HyperRaidBot installed.
2. Click the `ðŸ”— Link / Manage X Account` button.
3. Click the `[View Linked Accounts ðŸ”—]` button.
4. Press the <Icon icon="ellipsis-vertical" /> button next to the X Account you want to unlink.
5. Press the `[Unlink]` button.
6. Congratulations! You've now unlinked your X Account from the bot.
