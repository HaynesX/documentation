## Delete a SmartReply Context via `/smartreply`
1. Enter the `/smartreply` command
2. Press the `[⚙️ Configure SmartReply]` button
3. Go to the `[SmartReply]` tab
4. Press the <Icon icon="ellipsis-vertical" /> button next to the SmartReply Context you want to delete.
5. Press the `[Remove]` button.
6. Congratulations! You've deleted a SmartReply Context.
