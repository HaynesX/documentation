## Edit a SmartReply Context via `/smartreply`
1. Enter the `/smartreply` command
2. Press the `[⚙️ Configure SmartReply]` button
3. Go to the `[SmartReply]` tab
4. Press the <Icon icon="ellipsis-vertical" /> button next to the SmartReply Context you want to edit.
5. Press the `[Edit]` button
6. Edit any of the fields you want to change.
7. Press the `[Edit Context]` button to save the changes.
8. Congratulations! You've edited a SmartReply Context and can now generate SmartReplies on active raids (after enabling via the Toggle).
