## Link Shared X Account via `/smartreply`
Once on the `/smartreply` page, you can authorize HyperRaidBot to link a Shared X Account that other admins in your Telegram group can use to send SmartReplies on X.

### Instructions

1. Type `/smartreply` within a Telegram Group that has HyperRaidBot installed.
2. Go to the `[Accounts]` tab
3. Press the `[Link Shared Account]` button.
4. Click the `Authorize on X` button.
4. Login on X, and authorize HyperRaidBot to use your X Account.
5. Once authorized, you'll be redirected back to the Linked Accounts page where you can see the X Account that was linked.
6. You, and other admins, are now able to send SmartReplies on X using the new Shared X Account! ðŸŽ‰
