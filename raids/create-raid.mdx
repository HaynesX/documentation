## Create a Raid with `/raid [URL]`
<Tip>Example: `/raid https://x.com/mellow0x69/status/1810329069784793095`</Tip>

### Start a Raid
> You can start a raid that is being set up by clicking the `[Start 🏁]` button in the setup menu.
<Warning>
Pressing the `[Start 🏁]` button will start the raid immediately and will cancel any ongoing raid. 
</Warning>

### Modifying Targets
> Targets can be modified by clicking the `[Set Targets 🎯]` button in the setup menu. Learn more about [Setting Targets](/raids/setting-targets) here.

### Queue a Raid
> You can enqueue a raid that is being set up by clicking the `[Queue 📥]` button in the setup menu. Learn more about [Managing the Queue](/queue) to see how the `/queue` command works.

### Cancel a Raid Setup
> You can cancel a raid that is being set up by clicking the `[Cancel ❌]` button in the setup menu.
