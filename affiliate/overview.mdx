## Become an Affiliate
Start earning by referring Telegram groups to HyperRaidBot Pro. Open HyperRaidBot in a private chat and type `/affiliate` to access your affiliate dashboard.

### Install HyperRaidBot
[Click here to install HyperRaidBot](https://t.me/HyperRaidBot)

### Affiliate Dashboard
The affiliate dashboard is a Telegram Mini App with three main sections:

#### 1. Codes
Create and manage your unique referral codes to share with Telegram group admins. Each code displays:
- Your Payout Percentage (your earnings from each Pro purchase)
- Discount Amount (savings for groups using your code)

#### 2. Referrals
Track all successful referrals from groups purchasing Pro Tier:
- View complete referral history
- Monitor earnings per referral
- View what groups used your code to purchase Pro Tier

#### 3. Payouts & Balance
Manage your earnings and withdraw funds:
- Request payouts to BSC wallet addresses
- Track payout status and transaction IDs
- Zero withdrawal fees
- View complete payout history

### For Group Admins
To use an affiliate's referral code:
1. Add HyperRaidBot to your Telegram group
2. Type `/pro` in your group
3. Enter the referral code during checkout
4. Enjoy the discount on your Pro subscription

<Note>
Affiliate earnings are automatically calculated and added to your balance when groups successfully purchase Pro Tier using your referral code.
</Note>
